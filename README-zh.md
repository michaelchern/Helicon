一个面向 Codex 协作的最小可运行 C 项目骨架。

## 快速开始（Windows / PowerShell）

```powershell
powershell -ExecutionPolicy Bypass -File scripts/build.ps1
powershell -ExecutionPolicy Bypass -File scripts/test.ps1
```
