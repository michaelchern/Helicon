## Directory Contracts

| Path | Contract |
| --- | --- |
| `include/helicon/helicon.hpp` | Public C++20 API for RHI facade, render graph, and errors. |
| `src/rhi.cpp` | C++ facade implementation over backend interfaces. |
| `src/render_graph.cpp` | Render graph v0: image/pass declaration, execution, RGBA8 readback. |
| `src/rhi_internal.hpp` | Internal backend contract; not public API. |
| `src/backends/vulkan/` | Vulkan backend implementation. Keep Vulkan handles private. |
| `src/shaders/` | Embedded shader data, currently builtin triangle SPIR-V. |
| `src/ast/`, `src/dsl/` | Reserved for future Helicon Shader DSL. |
| `examples/triangle_graph/` | Headless triangle example that writes `triangle_graph.ppm`. |
| `tests/` | C++ smoke and C++ render graph regression tests. |
| `scripts/` | Canonical automation entry points. |
| `docs/zh/` | Human-editable Chinese source docs. |
| `docs/` | Derived English agent context. |
| `third_party/` | Vendored code only with explicit justification. |

## Search Routing

| Task | Start Here | Then Check |
| --- | --- | --- |
| Understand architecture | `docs/architecture.md` | `include/helicon/helicon.hpp` |
| Public C++ API change | `include/helicon/helicon.hpp` | `src/rhi.cpp`, `src/render_graph.cpp`, tests |
| Render graph behavior | `src/render_graph.cpp` | `tests/render_graph_test.cpp` |
| Vulkan backend work | `src/backends/vulkan/` | `src/rhi_internal.hpp` |
| Shader data | `src/shaders/` | Vulkan backend pipeline creation |
| Example behavior | `examples/triangle_graph/` | `README.md` |
| Build failure | `docs/build.md` | `scripts/build.ps1`, `CMakeLists.txt` |
| Test failure | `docs/testing.md` | `scripts/test.ps1`, `tests/` |
| Sync docs | `docs/doc-sync.md` | `skills/sync-docs/SKILL.md` |

## Agent Behavior Rules

- If Chinese and English docs conflict, prefer Chinese and sync English.
- Keep the project pure C++20; do not add C sources or C ABI unless explicitly requested.
- Do not expose Vulkan handles through public APIs.
- Keep generated outputs out of source control.
